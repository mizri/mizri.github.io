设备像素、css像素、设备独立像素、设备像素比、Viewport

发表于 2017-08-04

在还是以PC为主的年代，这些概念作为前端开发可能不需要了解，而在如今移动大行其道的时候，这些概念是必须深入理解的，如果你现在是在做移动页面的开发，而且并不了解这些概念的时候，会被一些奇怪的现象所迷惑，比如你在不同手机上发现你的布局文字等等东西会一样，这是为什么呢？

分辨率

或许有人很奇怪，这分辨率有什么好说的，实际上分辨率是有两种，如果分不清楚，px与px就傻傻分不清楚了，一种是物理分辨率实际就是我们通常所说的分辨率，一种是逻辑分辨率，逻辑分辨率是css分辨率。

此像素非彼像素

我们通常说的分辨率为1920*1080中的像素(px)概念并不是我们css中的像素（px）概念，这两者是不一样的概念，只不过在pc端分辨率1px=css的1px，但这个两个px概念是不一样的，接着往下看你就会明白了。

设备物理像素

又称为设备像素（device pixel）或物理像素（Physical pixels）
是屏幕能够控制的最小显示单位，在屏幕上它是一个物理发光点，在同一个设备上，它的物理像素是固定的，这是在厂商出厂时就是设置好的，意思就是一个设备的分辨率是固定。比如我们通常所说的1920*1080，意思就是横向有1920个发光点，纵向有1080个发光点，他是无法通过代码去控制的，物理分辨率对应就是它

css像素

它是个抽象概念，用在逻辑上衡量像素，它独立设备，简单说就是我们在写css代码的时候px，这通常是pc端叫法，css像素与屏幕无关，但与浏览器窗口的可视区域有关，这意味着css像素并不一定与设备像素一一对应，它实际上是个相对单位，相对的是设备物理像素

设备独立像素dip或dp(device independent pixels)

又称为设备无关像素或逻辑像素，它也是个相对单位，也是相对设备物理像素，它囊括了css像素的概念，你可以理解为计算机屏幕坐标系中的一个点，他是可以由程序控制的虚拟像素，逻辑分辨率对应的就是它，所以接下来主要以设备物理像素与设备独立像素来讲解

两者的关系

上面已经简单的介绍了三者的概念，设备独立像素是相对设备物理像素的，那么他们之间的关系是如何的呢？在pc端页面没有缩放的时候，一个设备物理像素=一个设备独立像素，
然而在移动端就不一样了，在如今的移动终端上他们基本上是不一样的，要解释清楚那么首先就要提到ppi这个概念

ppi

ppi是每英寸内有多少个设备物理像素，
更直白点的说法就是像素密度（pixel density），ppi的值越高，意味着画质越清晰，看起来就会更细腻，既然和屏幕的实际尺寸有关，那么直接上公式

这个就是我们接下来要将的 viewport

首页我们举个例子，当我把body下的一个div宽度设置为35%，那么这个35%是相对什么的呢？css中的块级元素是默认宽度都为100%，每一个css宽度百分比是根据它父元素的宽度来计算的，所以刚才我们所说的div的35%是相对body宽度，而body默认是100%相对html，html宽度也是100%，那么html的100%相对什么的呢？html相对的是视口，在css标准文档中它被称为初始包含块，这个初始包含块是所有CSS百分比的根源。

三个视口

布局视口(layout viewport)

他与浏览器的屏幕无关，他是完全独立的，css会根据他来计算，并且来约束它。我们看一张图片

通常我们可用document.documentElement.clientWidth或获取到它的值

视觉视口（visual viewport）

它是用户正在看到的网站区域，用户可以通过缩放来操作视觉视口（与设备像素比有关），同时不会影响布局视口，布局视口仍保持在原来的宽度。
他可以通过window.innerWidth来获取

理想视口

默认情况下，一个手机或平板浏览器的布局宽度为768-1024像素。这对于手机的窄屏来说并不理想。换句话说，布局视口 的默认宽度并不是一个理想的宽度。因此理想视口被引进了。只有当网站是为手机准备的时候才应该使用理想视口。当要添加理想视口。

理想视口设置

到底应该用那个视口

  1. 在PC浏览器中，浏览器窗口就是约束你的css布局的视口（又称为初始包含块）。他决定了用户可以看到什么
  2. 在移动设备上，桌面视口被拆成了两个：布局视口会限制你的css布局，视觉视口会决定用户能看到什么
  3. 移动端浏览器还有一个理想视口，它是对于特定设备上特定的浏览器布局视口的一个理想尺寸
  4. 可以把布局视口的尺寸设置为理想视口，实际上，这就是响应式设计的基础

举个css例子，以下两个属性都依赖视口，那么是依赖那个视口呢？

  1. position: fixed 标准：盒子(position设为fixed)相对与视口的位置是固定不变的
  2. vw和vh单位是相对视口的百分比，width:25vw 意味着元素的宽度是视口宽度25%
